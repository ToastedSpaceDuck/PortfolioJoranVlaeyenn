Project Koyle: Redesigning a cover
band website for modern look
and feel

UX Research · UI Design · Wireframing 2026 · Unofficial Project · Koyle
1.

Summary

In this project, I redesigned the website for the cover band Koyle in two weeks. My role was UX/UI designer, with the goal of creating a modern, user-friendly site that better reflects the band's style and is easy to maintain. The old site was outdated, causing both fans and potential bookers to lose interest. I began by analyzing the current site. I then designed several concept sketches and two versions of wireframes. After user testing with five participants and making adjustments, I created a final design for desktop and mobile. The result is a fresh, visually striking website with clear calls to action for booking the band.

3.

Problems and Objectives

3.1 Problems

Visually outdated

An outdated website often creates a poor first impression and undermines user trust.

Outdated information

Outdated information causes confusion and undermines user trust.

3.2 Objectives

Based on these challenges, I realized that the goal of this project was to design a clear and appealing website that:

Showcases the band's identity.

Presents important information in a clear and organized manner.

Makes the booking process as simple as possible.

4.

Key insights

Unclear call to action

The "Book Us" button was unclear and not prominent enough. Users didn't immediately see how they could book the band.

Outdated content

Many pages contained old tour dates and lyrics. This made it difficult to find up-to-date information.

Poor mobile experience

The old site was not optimized for mobile devices. Elements did not align properly, and important buttons were displayed too small.

6.

Results

The design now provides Koyle with a modern, functional website and a proposed implementation plan. It is recommended that this concept be implemented as a WordPress website. WordPress is the most widely used CMS worldwide and enables the band to manage events and content without any technical knowledge.
